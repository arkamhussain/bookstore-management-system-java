# bookstore-management-system-java
A role-based Java Bookstore Management System built using OOP principles and file handling, featuring user authentication, book management, genre-based search, and structured data persistence.
Bookstore Management System (Java)
Overview

This project is a role-based Bookstore Management System developed using Java with Object-Oriented Programming principles and file handling. The system is designed to simulate a real-world bookstore environment with structured user access and efficient data management.

Problem Statement

Manual bookstore management processes can lead to data inconsistency, limited scalability, and inefficient record handling. This system provides a structured and automated solution.

Key Features

Secure Login System (Admin / Cashier roles)

Add, View, and Manage Books

Genre-Based Search Functionality

File Handling for Data Storage

Role-Based Access Control

Structured OOP Architecture

Technologies Used

Java

OOP Concepts (Encapsulation, Inheritance, Polymorphism, Abstraction)

File Handling

NetBeans IDE

System Architecture

The system follows Object-Oriented design principles and separates responsibilities using structured classes for user management, book management, and system control.

How to Run

Open project in NetBeans

Compile all classes

Run the Main class

Login using predefined credentials
